simple-ajax-demo by dorward

Support code for an introductory Ajax talk covering both client and server

Interesting pieces

This project has the code structured in to a number of directories. Since it is designed as a learning aid rather then something that is actually runnable, here are pointers to the most interesting bits of code.

The Server Side Code

demo.pl is responsible for getting some data from the database, deciding if the user wants HTML or JSON, running the data through something to convert it to the right format, and then returning the result.

The HTML

html.tt is the HTML template that loads the JS and populates the default set of data.

The JavaScript

jQuery/demo.js is responsible for polling demo.pl and adding any new data to the page.
